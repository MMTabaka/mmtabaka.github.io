1. About Me

I am a 2nd year international student from Poland at The University of Sheffield. My major is BSc Artificial Intelligence and Computer Science.

My story with web development started in the 1st year of my studies. The Web and Interent Technology module introduced me to the basics of HTML, CSS and JS.

Both the module and final assignment, which was creating your personal website, weren't ones of my favourites. When I started to digging deeper and getting better at achieving my design goals I was getting more convinced to front-end web development and eventually decided to completely focus on this branch.

I designed and built this website for three reasons. First of all, practice. I wanted instead of learning countless CSS rules put some of them into practice and see during the process what I still need to improve. Secondly, I wanted a place where I will be free to put all my work. Last but not least, I was curious if after many failures with my assignment website I eventually will be able to implement what I designed in the first place and finding solutions to constantly appearing problems. I can see now a big improvement in my CSS skills and I can't wait to make them even greater.

Wine Browser App

A GUI application that takes user's text input with different wines and their properties and applies sorting on the list based on chosen filters. It displays the sorted list in a user interface along with minimum, maximum and average properties as well as with histogram, which graphically shows proportion of wines based on selected property.

This assignment helped me better understand OOP concept as well as Java Swing framework and put everything into practice.

  • Java

Map Traversal Problem

A problem which required to accommodate a Branch and Bound algorithm to terrain map in order to return the shortest possible path between two points on mountain terrain with the least possible climbing. The second part of the project was to implement A* algorithm based on the code from previous part by including different heuristics in calculating the distance and then compare the efficiency of both algorithms.

This assignment taught me theoretical knowledge by practice and how to use a LaTeX report.

  • Java

Global Engineering Challenge Week Project

A week long team project. Our task was to find a solution to frequent power outages and low electrical coverage in Lobitos and Piedritas areas in northern cost of Peru. Our idea was to install small wind turbines connected power grid and car batteries. It would increase the coverage, generate enough energy for a housold and with enough amount of batteries also sell some of the energy to the national grid.

This project streghen my leadership skills, required me to work in different engineering field than my own as well as improved my presentational and organisation skills.
